SDPLR

Sam Burer released a special version of SDPLR specifically for MAXCUT problems. Below there are links to both the package and a custom parameter file I used for larger graphs. Inside the tarball there are instructions for how to compile the C code (it is a simple command) as well as information on the format of the graph, the parameter file, and how to run the command.

SBMethod

I could not find my source of SBMethod, but it looks like it is still available at this link. I do remember that for very large graphs I had to edit the source code to use floats with larger number of bits, but I doubt you will need to do this.
